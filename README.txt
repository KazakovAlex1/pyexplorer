# PyExplorer - Консольный файловый менеджер на Python

Простой, но мощный консольный файловый менеджер для управления файлами и директориями.

## Возможности

- Просмотр файлов с детальной информацией (размер, тип)
- Создание файлов и директорий
- Удаление с подтверждением и рекурсивной опцией
- Переименование файлов и папок
- Перемещение и копирование файлов
- Поиск по шаблонам (рекурсивный и нет)

Можно будет реализовать:
- Логирование действий в файл
- Команда undo для отметы последнего действия
- Сделать поиск по содержимому файлов

Архитектура:
- argparse с subparsers
- Модули: os, os.path, shutil, sys
